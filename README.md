# Many-to-Many Relationship with JPA, Hibernate & Spring Boot

## Project Description
This project demonstrates how to implement a **Many-to-Many relationship** using **JPA (Java Persistence API)**, **Hibernate**, and **Spring Data JPA**.

It shows how two entities can be associated through a **join table**, which is a common scenario in real-world applications such as:
- Students ↔ Courses
- Users ↔ Roles
- Products ↔ Categories

---

## Objective
The goal of this project is to:
- Understand how **Many-to-Many relationships** work in JPA
- Learn how to use **Hibernate annotations**
- Practice **Spring Data JPA repositories**
- Manage relational data efficiently

---

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL (or H2 Database)
- Maven
